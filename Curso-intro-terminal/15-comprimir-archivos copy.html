Comprimiendo archivos tar y zip

  • Comprimir: tar: se usa mucho en repositorios, ej: tar -cvf < nombre del archivo.tar > < directorio donde se encuntran los archivos a comprimir >
  • Comprimir Gzip: tar: opcion de comprecion de Gzip, ej: tar -cvzf < nombre del archivo.tar > < directorio donde se encuntran los archivos a comprimir >
  • descomprimir: tar -xzvf < nombre del archivo comprimido >.tar.gz
  • Instalar Zip: apt install zip
  • Comprimir Zip: zip -r < nombre archivo >.zip + < nombre directorio >
  • Descomprimir Zip: unzip < nombre archivo >.zip
