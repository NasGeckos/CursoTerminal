Comandos de busqueda

  • which: wich < nombre >, busca en todas las rutas del PATH
  • find: find ./ -name file, busca los archivos file, en todos los directorios que tenga
  • find ./ -name *.txt: este busca todos los TXT
  • find ./ -type d -name Documents: despues de -type la D, hace referencia a Directorios, con fd busca files y directorios.
  • find ./ -size 20M: busca archivos mayores a 20mb.
