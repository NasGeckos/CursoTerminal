Manipular archivos y directorios:

  • mkdir: crear directorio, es posible crear varios al mismo tiempo, ej: mkdir dir1 dir2 dir3 dir4
  • touch: crear archivo, tambien admite creacion de multiples archivos en una linea, se puede agregar la extencion, ej: touch index.html
  • rm: borrar ARCHIVOS, ATENCION, no se puede recuperar luego de borrarlo. Con el comando rm -i, te pregunta si quieres borrar tal archivo.
  • rm -r: para borrar directorios
  • rm -ri: borrar directorios y archivos de forma interactiva, de esta forma ingresa al directorio y va preguntando archivo por archivo para finalizar con el directorio.
  • cp: copiar archivo, ej: cp file1 NOMBRE-DE-LA-COPIA
  • mv: mover archivo, ej: mv file1 .., es posible continuar con .. hasta donde quieras dejar el archivo
  • mv: permite renombrar un archivo, ej: mv file1 fileCopy <-- reenombra a este ultimo
  • tree -L: este muestra de una forma mas interactiva los directorios y archivos que estan dentro, agregando un numero despues de -L, indica la cantidad de nieveles a desplegar, ej: tree -L 2
