Explorando el contenido de nuestros archivos:

  • head: permite abrir el archivo en la consola, ej: head index.html, añadiendo -n 10, al final podremos visualizar las primeras 10 lineas de codigo, ej: head index.html -n 10
  • tail: muestra las ultimas lineas del archvo, tambien admite el -n NUM
  • less: permite visualizar el archivo de una forma mas interactiva, permite realizar busquedas dentro de archivo atravez de /palabra-clave, para salir del visualizador tecla Q
  • xdg-open: para abrir con un programa externo un archivo, ATENCION para matar la ejecucion desde la terminal es ctrl + C, sino no va a ser posible continuar usando la terminal.
  • nautilus: abre el directorio que indicamos con el navegador de carpetas de Windows, se debe matar el proceso (ctrl+C)
