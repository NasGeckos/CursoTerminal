Manejo de procesos

Matar, visualizar y manejar o filtrar procesos en la terminal

  • ps: procesos o comandos corriendo en la terminal actualmente
  • kill + < numero de proceso o PID >: termina el proceso
  • top: muestra una lista de los procesos que mas recursos consumen
  • cat > minota.txt: creamos una nota
  • ctrl+Z: suspendemos el proceso
  • jobs: nos muestra los proceso del foreground, con un numero de trabajo, que no es lo mismo que el PID
  • fg 1: para continuar con el editor de texto que cerramos repentinamente
  • ctrl+D: para guardar lo ingresado en la nota
  • ejecutamos jobs: y esta vez ya no nos muestra el Cat ejecutandoce en el background
