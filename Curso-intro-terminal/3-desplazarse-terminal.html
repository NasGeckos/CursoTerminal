Curso de Introducción a la Terminal y Línea de Comandos

Link al curso

Comando utiles para la consola:

  • clear: borra el texto de pantalla, shortcut: ctrl + L
  • ls: muestra un listado del contenido de la carpeta, con ls -lh, muetra los archivos en formato long (peso,fecha)
  • ls -la: muestra archivos ocultos
  • ls -S: ordena los directorios por tamaño
  • pwd: nos dice la ruta en la que estamos
  • cd: nos lleva a home
  • Operadores de ruta relativa: . y ..
    • Ruta situada: /home --- Ruta a la que queremos llegar: /home/codevars/Documents/Devs
    • cd ./ : cd ./Documents/Dev
    • cd .. : nos vuelve un directorio atras
  • file: para saber que tipo de archivo es, ej:
    • file Example.md ---> Example.md: UTF-8 Unicode text
    • ej con ruta relativa: file ./Pictures/screenshot.png ---> ./Pictures/screenshot.png: PNG image data, 1920 x 1080, 8-bit/color RGBA
