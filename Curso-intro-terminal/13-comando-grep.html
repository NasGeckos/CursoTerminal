Usando el comando grep

Grep, nos permite econtrar concidencias de una busqueda dentro de un archivo de texto

  • grep: ej: grep < palabra clave a buscar > movies.csv
  • grep -i: ignola el case sensitive, es decir si es minuscula o mayuscula
  • grep -c: va a contar el numero de ocurrencias que hay en el archivo
  • grep -v: cuenta las que NO coinciden
  • grep -vi towers movies.csv > sintowers.txt : guarda la busqueda de pelicuas sin la palabra towers ignorando si es minuscula o mayuscula, en el archivo sintowers.txt
  • wc: contador de palabras, ej: wc movies.csv, devuelve: lineas de archivo, cuantas letras, numero de bits
  • wc: wc -l: numero de lineas, wc -w: numero de palabras, wc -c : peso en bits
