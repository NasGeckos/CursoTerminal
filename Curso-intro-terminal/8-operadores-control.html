Encadenando comandos: operadores de control

  • ;: Ejecuta de manera sincrona, ej: ls; mkdir holi; cal --> ejecuta un comando tras otro.
  • &: Ejecuta de manera asincrona, ej: ls & date & cal --> ejecunta un comando por hilo o segundo planto, o por Shell. ls lista de archivos, date wed, mar, 31, hora, año, cal calendario
  • &&: Ejecutar de manera condicional, ej: mkdir test && cd test, crea el directorio test y luego nos posiciona en la carpeta test, solo si el primer condicionante se ejecuto de manera adecuada.
  • ||: Operador OR, ejecuta la cadena por bloques, siempre y cuando se pueda, si no, sigue con la proxima cadena, ej: cd asdasdasd || touch archivo.txt || echo "Archivo creado", de esta forma va a dar un error por que el directorio no existe, continua creando el archivo.txt y luego imprime por pantalla echo ...
