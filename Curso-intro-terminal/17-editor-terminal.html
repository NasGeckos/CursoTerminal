Editores en la terminal

Explorar editor de texto VIM

  • vim: para ejecutarlo, :q+Enter, para salir
  • vim index.html: crear archivo de texto en vim
  • i: modo Insert, precionandola nos deja empezar a editar el archivo
  • Esc: para salir del modo Insert
  • /titulo: para buscar una palabra
  • d+d: estando en el modo normal borrar una linea
  • :w : para guardar el documento
  • :wq : para guardar y salir
  • wq! : fuerza la salida
